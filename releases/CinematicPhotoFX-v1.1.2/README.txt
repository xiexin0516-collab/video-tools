Cinematic Photo FX v1.0.0
电影级图片特效工具

版本: 1.0.0
构建日期: 2025-09-06 09:15:17
构建环境: Windows 10/11 (64-bit)

功能特色:
- 电影级图片特效处理
- 多种视觉效果 (推拉、移动、旋转、滤镜等)
- 智能特效组合推荐
- 多语言支持 (中文/英文)
- 批量图片处理
- 高质量视频输出

系统要求:
- Windows 10/11 (64位)
- 无需安装，解压即用

使用方法:
1. 双击 CinematicPhotoFX.exe 启动程序
2. 选择要处理的图片文件
3. 选择特效组合
4. 配置视频参数
5. 开始生成视频

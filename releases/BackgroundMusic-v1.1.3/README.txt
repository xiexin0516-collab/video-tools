背景音乐生成器 v1.0.0 
 
功能特点: 
- 支持4种音乐风格：神秘悬疑、平静舒缓、紧张氛围、希望向上 
- 高级旋律引擎：基于调式音阶和马尔可夫链 
- 管弦级编曲：包含弦乐、钢琴、鼓点、环境音效 
- 支持中英文界面切换 
- 可自定义音乐时长和输出文件名 
 
使用方法: 
1. 双击运行BackgroundMusic.exe 
2. 选择音乐风格 
3. 设置时长和文件名 
4. 点击"开始生成"按钮 
5. 等待生成完成 
 
系统要求: 
- Windows 10/11 (64位) 
- 无需安装Python或其他依赖
